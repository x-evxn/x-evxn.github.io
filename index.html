Cryptix

Cryptix is a simple, user friendly file encryption and decryption tool built using Chat-GPT and Python. The app uses AES (Advanced Encryption Standard) encryption in CBC (Cipher Block Chaining) mode to securely encrypt and decrypt files. The app also utilizes PBKDF2 (Password-Based Key Derivation Function 2) to derive a secure encryption key from a passphrase provided by the user. You can download the tool, and read more at Evxn's Github Repository.
